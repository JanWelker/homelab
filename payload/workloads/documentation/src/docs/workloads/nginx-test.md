# nginx-test

A simple Nginx application for testing deployment flows.

## Components

- **Deployment**: `payload/workloads/nginx-test/deployment.yaml`
- **Service**: `payload/workloads/nginx-test/service.yaml`
- **HTTPRoute**: `payload/workloads/nginx-test/httproute.yaml`
- **NetworkPolicy**: `payload/workloads/nginx-test/networkpolicy.yaml`

## Usage

This application is commonly used to verify that the ingress/gateway-api configuration is working correctly.
